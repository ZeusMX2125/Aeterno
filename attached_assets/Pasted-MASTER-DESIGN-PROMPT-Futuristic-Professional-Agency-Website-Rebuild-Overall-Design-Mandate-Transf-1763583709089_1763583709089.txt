MASTER DESIGN PROMPT: Futuristic Professional Agency Website Rebuild

Overall Design Mandate: Transform the current agency website into a cohesive, high-end, futuristic, and professional showcase. Prioritize elegant dark UI, subtle dynamism, unified visual language, and clear user journeys. Every element should feel premium and intentional.

GLOBAL STYLING & CORE AESTHETICS:
Color Palette:

Primary: #000000 (Pure Black) for background.

Accent: #FF4500 (Safety Orange) - used for primary CTAs, active states, subtle glows, and key text highlights.

Secondary Text/Details: White (#FFFFFF) and light grey (#CCCCCC) for readability.

Glassmorphism Effect: A dark, semi-transparent background (e.g., rgba(18, 18, 18, 0.6)) with a backdrop-filter: blur(10px); and a very subtle 1px white or light grey border (rgba(255, 255, 255, 0.1)).

Typography:

Font Family: Use a modern, highly legible variable sans-serif font (e.g., Inter or Plus Jakarta Sans from Google Fonts) to allow for precise weight control.

Headlines: Bold, strong weights for impact.

Body Text: Optimized for readability on a dark background.

Subtext/Descriptions: Slightly increased size for improved legibility compared to current.

Background Dynamic (Hero & Select Sections):

Moving Futuristic Gradient: Implement a subtle, slow-panning, multi-directional gradient background, primarily within the black/dark grey spectrum with hints of the Safety Orange or a deep blue/purple. This should be achieved via CSS keyframe animations on a large background element (e.g., a ::before pseudo-element on the hero section). The movement should be almost imperceptible until consciously observed, adding depth without distraction.

Subtle Noise Texture: Overlay a highly transparent (3-5% opacity) grainy noise texture (background-image: url('noise.png');) across the entire site to prevent banding and add a premium, film-like quality.

Interaction Design (Hover/Active States):

All interactive elements (buttons, cards, links) should have smooth, subtle transitions and hover effects.

Hover effects should utilize the Safety Orange for highlights, text color changes, or soft glows.

HEADER & NAVIGATION:
Top Navigation Bar (New Addition):

Persistent & Transparent: A discreet, transparent navigation bar fixed to the top of the viewport.

Left: Agency Logo (clean, monochromatic version of current logo).

Right: Primary Navigation Links (e.g., "Services," "Work," "About," "Contact") and a prominent "Let's Talk" button (Safety Orange fill, white text).

Hover Effect: Links subtly glow orange on hover.

Service Ticker:

Keep the existing ticker style.

Positioning: Ensure it sits logically, providing immediate context below the main hero headline.
