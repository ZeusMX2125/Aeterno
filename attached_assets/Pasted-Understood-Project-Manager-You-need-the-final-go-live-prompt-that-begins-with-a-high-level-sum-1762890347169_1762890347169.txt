Project: Aeterno Media Landing Page

Goal: The primary goal is lead generation. We are not building a simple, static website. We are building a single-page, narrative-driven "Impress-to-Quote" funnel.

The "Wow" Factor: The user's journey is a story. As they scroll, the page will "lock" (pin) at key sections to showcase a "wow" animation. This will tell a 3-part story about the agency's core services.

Impress: The user is wowed by the "scroll-and-lock" animations.

Inform: The user sees all 8 of the agency's services.

Convert: The user clicks a "Get a Quote" button, which opens an interactive, multi-step modal form (not a separate page) to schedule a consultation.
